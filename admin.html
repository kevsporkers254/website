Admin Login

Enter a GitHub fine-grained token with contents read/write access to kevsporkers254/website. The token stays in this browser tab only and is not saved into the website.

How to create the token
  1. Open GitHub Settings, then Developer settings.
  2. Create a fine-grained personal access token.
  3. Select repository access for kevsporkers254/website.
  4. Give Contents permission: read and write.
  5. Copy the token and paste it here when you need to edit.
